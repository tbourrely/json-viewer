# json-viewer

### Getting started

```
git clone https://github.com/tbourrely/json-viewer.git
cd json-viewer
pnpm i
pnpm run dev
```

### Running tests

```
pnpm run test
```

### Running linters

```
pnpm run lint
```

### Adding json files

Simply add json files in the *assets* directory.

> example json files were generated through ChatGPT.
